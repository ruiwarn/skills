# RN7326 / RN202x 用户手册 Skill

## 作用
用于回答 RN7326 / RN202x 三相计量 SoC 的寄存器、初始化流程、通信接口、时序、电气特性、错误码等问题。
回答必须以 references/manual.md（或 manual.pdf）为准，并给出可追溯的引用位置（章节标题/小节名；如果你 MD 里保留了页码则给页码）。

## 何时触发
- “寄存器/位定义/默认值/访问权限”
- “I2C/SPI/串口等接口时序、波特率、帧格式”
- “启动/校准/计量计算流程”
- “电气特性/典型应用电路/引脚复用”
- “中断/错误码/故障处理”
- 用户要求“按手册原文/按章节说明”

## 检索方法（强制）
1. 先在 references/manual.md 搜索关键词：寄存器名、地址(0x..)、信号名、模块名、错误码、表名。
2. 找到对应章节/表格后，再组织答案。
3. 输出结构：
   - 结论（简洁）
   - 依据（手册中的章节/表格/段落引用）
   - 工程建议/注意事项（如果手册未明确则标注“推断”）

## 引用格式（建议）
- 引用时写：`来源：manual.md / <章节标题> / <关键段落首句>`  
- 如果你的 MD 内含 “Page N” 或 “第 N 页” 标记，则同时写页码。
